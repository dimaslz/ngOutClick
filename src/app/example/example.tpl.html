How to use

ngOutClick is available on Bower, so you can execute the next command:

bower install ngOutClick

and inject module on your controler: angular.module('myModule', ['ngOutClick']);
Now, you can use this directive like following examples.

Examples

Example with element DIV and content text

with init method

code
            <div
              ng-click="clickOnExample1()"
              ng-out-click
              out-click-callback="outClickChangeExampleText1Callback()"
              out-click-init="changeExampleText1Init()">
                Click on me for change the next text: "{{ textExample1 }}"
            </div>

result
Click on me for change the next text: "{{ textExample1 }}"

without init method

code
            <div
              ng-click="clickOnExample2()"
              ng-out-click
              out-click-callback="outClickChangeExampleText2Callback()">
                Click on me for change the next text: "{{ textExample2 }}"
            </div>
result
Click on me for change the next text: "{{ textExample2 }}"

Example with Dropdown

code
            <div ng-click="toggleEdit()" ng-if="showText">Click to edit with dropdown. Current option selected:
              <strong>"{{ currentOption.value }}"</strong>
            </div>
            <div class="btn-group" dropdown is-open="status.isopen" ng-if="!showText">
              <button
                class="btn btn-primary dropdown-toggle"
                dropdown-toggle ng-disabled="disabled"
                ng-out-click
                out-click-callback="toggleEdit()"
                out-click-init-count="true"
                type="button">
                  Button dropdown
                <span class="caret"></span>
              </button>
              <ul class="dropdown-menu" role="menu">
                <li ng-click="selectOption(option)" ng-repeat="option in options">
                  {{ option.value }}
                </li>
              </ul>
            </div>
result
Click to edit with dropdown. Current option selected: "{{ currentOption.value }}"
  • {{option.value}}
